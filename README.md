Discord Token Extractor, tarayıcı üzerinden açık olan Discord hesabınızın tokenini saniyeler içinde, 0 hata ile çeken ve panoya kopyalayan modern bir Chrome/Edge eklentisidir. Gelişmiş "Main World Injection" tekniği kullanarak Discord'un güncel LocalStorage korumalarını atlatır.

## ✨ Özellikler

- **Güncel Bypass Mantığı:** Discord'un "Isolated World" korumasını aşarak kodu direkt ana dünyaya (`world: "MAIN"`) enjekte eder.
- **Tek Tıkla Kopyalama:** Tokeni bulur bulmaz anında panoya kopyalar.
- **Modern Karanlık Tema:** Göz yormayan, premium hissiyatlı neon mor ve lacivert (Dark Mode) UI tasarımı.
- **Hafif ve Hızlı:** Gereksiz hiçbir arka plan servisi çalıştırmaz. Sadece butona bastığınızda işlem yapar.
- **Manifest V3 Uyumluluğu:** Google'ın en son eklenti standartlarına göre kodlanmıştır.

## 🛠️ Kurulum Adımları

Bu eklenti şu an için Chrome Web Store'da bulunmamaktadır, manuel olarak yüklemeniz gerekmektedir:

1. Bu repoyu bilgisayarınıza ZIP olarak indirin (`Code -> Download ZIP`) ve bir klasöre çıkartın.
2. Google Chrome veya Brave/Edge tarayıcınızı açın.
3. Adres çubuğuna `chrome://extensions/` yazarak Eklentiler sayfasına gidin.
4. Sağ üst köşeden **"Geliştirici Modu" (Developer Mode)** seçeneğini aktif hale getirin.
5. Sol üstte beliren **"Paketlenmemiş öğe yükle" (Load unpacked)** butonuna tıklayın.
6. İndirip zip'ten çıkardığınız klasörü seçin.
7. Eklenti kuruldu! Sağ üstteki yapboz 🧩 ikonundan eklentiyi sabitleyebilirsiniz.

## 🚀 Nasıl Kullanılır?

1. Tarayıcınızdan [discord.com/app](https://discord.com/app) adresine gidin ve hesabınıza giriş yapın.
2. Sağ üstten **Token Extractor** eklentisine tıklayın.
3. **"Tokenimi Getir"** butonuna basın.
4. Tokeniniz ekranda belirecek ve otomatik olarak panonuza (Clipboard) kopyalanacaktır!
